# Welcome to your product's documentation

You can start by going to the [Getting Started](https://indiekit.pro/app/docs) page.

Docs engine is  powered with [Fumadocs](https://fumadocs.vercel.app/) with pre-installed features.


Just head over to `src/content/docs` and start writing your documentation.


## Pre-installed features:

- [x] Automatic icon generation (check sidebar icons, they are powered by [Lucide](https://lucide.dev/))
- [x] Automatic [sitemap](/sitemap.xml) generation
- [x] Search feature
- [x] Automatic OG Image generation
- [x] Copy as Markdown feature
